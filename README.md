# Chat
Chat сделан специально для практики МГОК, чтобы люди могли обмениваться сообщениями

## Установка и запуск
  ```cmd
git clone https://github.com/impovskii/chat-api
```

  ```cmd
cd server
npm run start
```

  ```cmd
cd client
npm start
```

## Технический стек
- Node.js
- React + Vite.js
- Socket.io
- Express.js
